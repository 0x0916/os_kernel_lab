# ucore for riscv

> A 32-bit teaching Operating System on RISC-V.

## What is it

[ucore](https://github.com/chyyuu/ucore_os_lab/tree/riscv32-priv-1.10)是清华大学计算机系本科操作系统课程的教学操作系统[ucore_os_lab](https://github.com/chyyuu/ucore_os_lab)在RISC-V架构上的移植。
